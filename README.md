# 智能社区停车场管理系统 java497

## 项目概述


## 技术栈

### 开发环境
- JDK 1.8
- Tomcat 8.0
- MySQL 8
- Redis 6

### 前端技术
- Vue 2
- Element UI
- ECharts
- BaiduMap

### 后端技术
- Spring Boot
- Spring MVC
- MyBatis Plus
- Spring Data Redis
- Swagger

### 第三方服务
- 腾讯OCR文字识别接口
- 百度地图接口
- ECharts 图表接口

## 功能模块

### 车场管理
- 物业公司管理
- 停车场管理
- 车辆管理
- 停车场地图

### 停车管理
- 车牌识别停车记录
- 缴费记录

### 系统管理
- 用户管理
- 角色管理
- 菜单管理
- 日志管理

## 系统角色
- 系统管理员
- 物业管理员
- 停车场工作人员
- 停车用户

## 运行环境
- 服务器：推荐使用 Linux 服务器
- 数据库：MySQL 8 或兼容版本
- 缓存：Redis 6 或兼容版本

## 部署步骤
1. 环境搭建：部署 JDK、Tomcat、MySQL、Redis
2. 数据库初始化：导入系统数据库结构
3. 应用部署：部署后端应用到 Tomcat
4. 配置第三方服务：配置腾讯OCR与百度地图接口
5. 启动服务：启动 Tomcat，并访问前端页面

## 目录结构
```
.
├── backend                # 后端代码目录
├── frontend               # 前端代码目录
├── database               # 数据库脚本目录
└── deployment_scripts     # 部署脚本目录
```

## 核心亮点
- 利用大数据分析，提供停车场使用情况统计，助力物业高效管理
- 采用前后端分离架构，易于维护和扩展
- 系统具备完善的角色权限管理，确保数据安全
- 实时监控系统日志，便于问题追踪与系统优化
- 基于百度地图接口，提供停车场地图导航服务
